# UNO Admin Delete Feature

Added:
- /api/admin-list
- /api/admin-delete
- Delete section inside /admin.html

After deploy:
1. Open /admin.html
2. Enter ADMIN_SECRET
3. Click Load Recent Pixels
4. Delete the pixel you want

Note: This deletes the database row. The image file may remain in Supabase Storage unless you delete it manually.
